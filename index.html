Ruby

Ruby is a language for people who like things to make sense. It uses language very similar to every day usage, rather that adding a ton of punctuation to trip you up! So as you might imagine, a lot of ruby programmers are happy with it and don't want to change. If you want to know more about Ruby straight from the Ruby community, click here.

JavaScript

JavaScript is a language for people who have an obsession with detail. This language requires very precise grammar in order to ensure function. One missing semicolon has been known to take years off of a programmer's life. But if you enjoy hunting for characters lost in a sea of characters, then this is the language for you. You will spend countless hours searching for an elusive parenthesis or semicolon. To learn more about JavaScript, check out a community resource page here.

Python

Python is considered a general-purpose language and is for people who want to code in many different types of applications! Python programmers tend to be well-rounded and even-tempered folks. If this sounds interesting to you, then click here to learn more.
